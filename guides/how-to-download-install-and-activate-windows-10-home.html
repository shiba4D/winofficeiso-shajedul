How to Download, Install, and Activate Windows 10 Home

Setting up Windows 10 Home 32/64-bit? This guide shows you how to download the installation files, install from USB or DVD, and activate your license key — step by step.

✅ Before You Begin

  • 1 GHz processor or faster
  • 2 GB RAM (32-bit) / 4 GB RAM (64-bit)
  • 20–32 GB available storage
  • UEFI/BIOS with bootable USB or DVD support
  • Internet access for activation

💡 Step 1: Download the Media Creation Tool

👉 Download Media Creation Tool
  1. Run the tool as Administrator
  2. Select Create installation media for another PC
  3. Choose Language, Edition: Windows 10, and Architecture
  4. Choose USB flash drive or ISO file to burn later
  5. Click Finish when done

💻 Step 2: Install Windows 10 Home

  1. Insert USB or DVD into the PC
  2. Restart and access boot menu (F2, F12, Delete, Esc)
  3. Boot from USB/DVD
  4. Set your language, time, and input
  5. Click Install Now and enter your product key
  6. Accept license terms, click Next
  7. Select Custom: Install Windows only
  8. Choose drive/partition, format if needed, and click Next

🧠 If your PC boots into the old OS, shut it down fully before retrying boot from USB/DVD.

🔐 Step 3: Activate Windows 10 Home

  1. Go to Settings → Update & Security → Activation
  2. Click Change product key
  3. Enter your 25-character license key
  4. Click Activate

✅ Once successful, Windows will display “Windows is activated.”

❓ Troubleshooting Activation

  • Check internet connection
  • Double-check for typos or missing characters
  • Reboot and try activation again
  • Contact support if issues persist
